# chat_funst_bot

Telegram Business Bot с интеграцией Funstat API.

## Возможности

- Уведомление о новых контактах с карточкой пользователя (Telegram + Funstat)
- /spam — авторассылка сообщений
- /troll — троллинг с таймером (авто-стоп при блокировке)
- /mute / /unmute — заглушить чат
- /mirror — зеркальный режим
- /save — сохранить сообщение
- /voice, /music, /sound, /quran — аудио-команды
- /voicemod — изменение голоса
- /ai, /groq — AI-ответы
- /q — цитаты
- /search — поиск
- /secret — секретные сообщения

## Переменные окружения (Railway)

| Переменная | Описание |
|---|---|
|  | Токен Telegram-бота |
|  | Telegram ID владельца |
|  | Токен Funstat API |
|  | GitHub репо для хранения стейта (опционально) |
|  | GitHub токен для стейта (опционально) |
